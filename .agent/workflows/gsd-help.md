# GSD Command Reference

**GSD** (Get Shit Done) creates hierarchical project plans optimized for solo agentic development.

## Core Setup
- `/gsd-new-project` - Initialize everything
- `/gsd-map-codebase` - Analyze existing code

## Planning & Execution
- `/gsd-discuss-phase {N}` - Clarify vision
- `/gsd-plan-phase {N}` - Create PLAN.md files
- `/gsd-execute-phase {N}` - Run the plans

## Iteration & Maintenance
- `/gsd-progress` - Current status check
- `/gsd-transition` - Move to next phase
- `/gsd-verify-work {N}` - User testing flow
- `/gsd-debug {issue}` - Scientific debugging

## Roadmap Evolution
- `/gsd-add-phase {desc}` - Append new phase
- `/gsd-insert-phase {N} {desc}` - Insert decimal phase
- `/gsd-remove-phase {N}` - Delete future phase

## Todo Management
- `/gsd-add-todo {desc}` - Capture idea
- `/gsd-check-todos` - Review and work on ideas

## Global Settings
- `/gsd-settings` - Toggle agents & profiles
- `/gsd-update` - Keep GSD version current
